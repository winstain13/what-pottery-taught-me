# What Pottery Taught Me

A single-page site exploring the stages of pottery and the lessons of embracing the unexpected.

## Viewing Locally

Open `index.html` in your browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`

## Content

The page covers all eight stages of pottery:
1. Preparing the Clay
2. Forming (Wheel Throwing)
3. Trimming
4. Drying
5. Bisque Fire
6. Glazing
7. Glaze Fire
8. Buffing & Finishing

Each section includes the stage duration and detailed steps.

## Design

- Responsive layout (mobile and desktop)
- Timeline visualization showing greenware → bisqueware → finished product
- Clean, modern aesthetic
- Smooth scrolling sections
